# Proximity Green Platform

Workshop17 admin platform — SvelteKit + self-hosted Supabase + Trigger.dev on Hetzner/Coolify.

## Docs

- [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) — stack, layout, data flow, deployment
- [`docs/CONVENTIONS.md`](docs/CONVENTIONS.md) — naming, services, UI primitives, migration rules

## Quick start

```sh
cd apps/web
npm install
npm run dev
```

Production: `poc.proximity.green` (deploy via SSH — see `ARCHITECTURE.md`).
